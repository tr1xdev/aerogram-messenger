# Features

### Messaging

* **Real-time Communication:** Private chats, group conversations, and channels.
* **Rich Media:** Support for file attachments and images with zoom capabilities.
* **Content Control:** Message editing, deletions, and reply threads.
* **Previews:** Rich previews for links and text content.

### Identity & Security

* **Authentication:** Secure session management and OTP-based registration.
* **Presence:** Real-time online/offline status and last-seen activity.
* **Profile Management:** Customizable user avatars and account settings.

### Chat Management

* **Group Admin Tools:** User management including kicking, promoting, and demoting members.
* **Organization:** Pinning and removing chats from the active list.
* **Interface:** Clean, intuitive UI designed for efficient navigation.

### Platform & Extensibility

* **API-first:** Rate-limited endpoints for high performance and stability.
* **Bot SDK:** Extensible system for custom bot development and integration.
* **Bot Ecosystem:** Dedicated interface for managing and tracking active bots.
