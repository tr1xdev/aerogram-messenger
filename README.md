# Aerogram

Aerogram is an open-source full-stack web application focused on security and privacy. The backend is written in [Go](https://go.dev/) with a [GraphQL](https://graphql.org/) API, backed by [PostgreSQL](https://www.postgresql.org/) and [Redis](https://redis.io/). The frontend uses React and TypeScript with [Tailwind CSS](https://tailwindcss.com/) and [Shadcn UI](https://ui.shadcn.com/), communicating with the backend via GraphQL and WebSocket for real-time updates.
